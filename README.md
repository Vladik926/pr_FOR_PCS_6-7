3.Отчет о выполненной работе:

Была успешно реализована верстка ключевых экранов мобильного приложения, включая:

Экран регистрации нового аккаунта

Экран авторизации с полями для ввода логина и пароля

Главную страницу интернет-магазина с товарами

Раздел "Избранное" с понравившимися товарами

Корзину с выбранными товарами

Использованные элементы интерфейса: В процессе разработки применялись различные виджеты Flutter: текстовые элементы (Text), карточки товаров (Card), поля ввода (TextField), кнопки (ElevatedButton, IconButton), элементы навигации (BottomNavigationBarItem), изображения (Image), а также списки товаров в виде сетки (GridView) и интерактивные элементы (GestureDetector).

Особенности реализации: В ходе работы удалось воссоздать все основные функциональные экраны согласно требованиям. Единственным отличием от первоначального макета стало отсутствие сплеш-экрана (экрана-заставки), что связано с техническими сложностями при импорте и отображении логотипа приложения. Все остальные элементы интерфейса соответствуют поставленным задачам и обеспечивают удобный пользовательский опыт.
